# Какая техника нужна в твоей области интересов

> Страница обновлена автоматически по локальным ответам Wikidata. Это черновик, который удобно потом доработать вручную.

## О чём эта статья

Почему дизайнеру, музыканту, программисту и геймеру нужны разные устройства.

## Что нашлось в Wikidata

- **смартфон** — найдено по запросу для **smartphone**. [Wikidata](https://www.wikidata.org/entity/Q22645)
- **компьютер** — найдено по запросу для **computer**. [Wikidata](https://www.wikidata.org/entity/Q68)
- **ноутбук** — найдено по запросу для **laptop**. [Wikidata](https://www.wikidata.org/entity/Q3962)
- **battery**. [Wikidata](https://www.wikidata.org/entity/Q2891692)

## Связи, которые видны в данных

- **компьютер** → **не путать с** → **вычислительная техника**
- **смартфон** → **не путать с** → **карманный персональный компьютер**
- **компьютер** → **относится к вики-проекту** → **Computer hardware task force**
- **смартфон** → **подкласс от** → **мобильный телефон**

## Что это даёт для текста

По этим данным можно опереться на реальные сущности и связи вокруг темы **Какая техника нужна в твоей области интересов**. В черновике статьи полезно объяснить, как понятия **техника, интересы, выбор, задачи** связаны друг с другом и почему это важно в повседневной цифровой жизни.

## Что ещё посмотреть в данных

- В обратных связях понятие **компьютер** встречается как узел с отношением **это частный случай понятия** не меньше 55 раз.
- В обратных связях понятие **компьютер** встречается как узел с отношением **элемент для значения** не меньше 52 раз.
- В обратных связях понятие **компьютер** встречается как узел с отношением **подкласс от** не меньше 33 раз.
