## О чём эта статья

Как понять, какой контент безопасно публиковать, а какой лучше оставить при себе.

## Что нашлось в Wikidata

- **интернет-травля** — найдено по запросу для **cyberbullying**. [Wikidata](https://www.wikidata.org/entity/Q494348)
- **персональные данные** — найдено по запросу для **personal data**. [Wikidata](https://www.wikidata.org/entity/Q3702971)
- **социальная сеть** — найдено по запросу для **social media**, через связь **subclass of**. [Wikidata](https://www.wikidata.org/entity/Q3220391)
- **personal data**. [Wikidata](https://www.wikidata.org/entity/Q63981718)

## Связи, которые видны в данных

- **интернет-травля** → **общая категория по теме** → **Category:Cyberbullying**
- **интернет-травля** → **подкласс от** → **травля**
- **персональные данные** → **общая категория по теме** → **Категория:Персональные данные**
- **personal data** → **главный регулирующий текст** → **Общий регламент по защите данных**

## Что это даёт для текста

По этим данным можно опереться на реальные сущности и связи вокруг темы **Что можно и нельзя постить**. В черновике статьи полезно объяснить, как понятия **пост, публикация, безопасность, фото** связаны друг с другом и почему это важно в повседневной цифровой жизни.
