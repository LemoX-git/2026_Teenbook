# Страх пропустить что-то (FOMO)

> Страница обновлена автоматически по локальным ответам Wikidata. Это черновик, который удобно потом доработать вручную.

## О чём эта статья

Почему кажется, что все вокруг живут интереснее, и как не попасть в ловушку сравнения.

## Что нашлось в Wikidata

- **Боязнь пропустить интересное** — найдено по запросу для **fear of missing out**. [Wikidata](https://www.wikidata.org/entity/Q16753510)
- **virtual community**. [Wikidata](https://www.wikidata.org/entity/Q14164801)
- **социальная сеть** — найдено по запросу для **virtual community**, через связь **subclass of**. [Wikidata](https://www.wikidata.org/entity/Q3220391)
- **social network**. [Wikidata](https://www.wikidata.org/entity/Q17164759)

## Связи, которые видны в данных

- **Боязнь пропустить интересное** → **подкласс от** → **тревога**
- **социальная сеть** → **не путать с** → **социальная сеть**
- **Боязнь пропустить интересное** → **противоположно** → **Joy of Missing Out**
- **social network** → **подкласс от** → **языковое сообщество**

## Что это даёт для текста

По этим данным можно опереться на реальные сущности и связи вокруг темы **Страх пропустить что-то (FOMO)**. В черновике статьи полезно объяснить, как понятия **FOMO, сравнение, соцсети, тревога** связаны друг с другом и почему это важно в повседневной цифровой жизни.
