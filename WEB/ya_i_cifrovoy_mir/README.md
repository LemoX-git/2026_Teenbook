# WEB/ya_i_cifrovoy_mir

Черновая структура финальной части раздела **«Я и цифровой мир»**.

Каждая подпапка соответствует одной теме раздела, а внутри папки `concepts/` лежат markdown-страницы будущей энциклопедии.

Темы:
- **Моя зависимость** → `moya_zavisimost`
- **Мои игры** → `moi_igry`
- **Моя информационная гигиена** → `moya_informacionnaya_gigiena`
- **Моя безопасность в сети, приватность, публикация** → `moya_bezopasnost_v_seti`
- **Моя реальность/виртуальность** → `moya_realnost_i_virtualnost`
- **Моя техника** → `moya_tehnika`
